# Timmick
## 概要
時間を計測することで、それぞれの場面に使用した時間を視覚化でき
また自分の成果を投稿できるアプリケーションです。

## Application URL

## ゲストユーザーログイン機能付き
未ログイン状態でのヘッダーに「ゲストユーザーログイン」をクリック
（ゲストユーザーの編集・削除はできません。）

## 利用方法
1.ホーム画面にて用意されている４つの分野（学習・リラックス・フィットネス・娯楽）で時間を計測し保存する
2.使用した時間と成果を投稿する
3.気になるユーザーをフォローし、投稿にコメント・いいねする
